# 独显

不要买a卡  
又不便宜性能还弱bug还多

n卡

- 现在主流是50系，5060、5070满功耗115w，5070ti满功耗140w，5080、5090满功耗175w  
  - 可以根据这一点去看笔记本的性能是不是满血，例如一个16寸总释放175w，cpu我们假设打游戏50w够用，175w-50w=125w>115w
- 后面数字是“50”的一般价格不会比”60“低多少，但是性能低不少，所以一般不考虑
- 2050、3050  
  - 这两是超低端游戏本可能会出现的，2050实际上是30系架构的烂卡，性能跟680m、780m一桌，千万不要买  
  - 3050看价格定，就我现在的时间，我认为3000左右可以考虑

# 核显

intel

- 12代架构的核显都是垃圾，现在不用考虑
- xe1也就是ultra100的，除非价格显著低于780m，不然最好别考虑
- xe1+，也就是ultra200H的，能买，但是我觉得不如78m
- xe2，ultra200v的，价格合适看着办，但是我不觉得它还会有合适价格
- xe3，ultra300H，无敌架构，12xe的xe3能摸3060m屁股。加上cpu进步，可以跟老3060游戏本掰掰手腕

amd

- 680m，常出现于zen3，除非真的没预算，不然可以不用考虑
- 780m，性价比之王，有一堆换皮u
- 880m、890m、860m，它们对应的cpu都是大小核zen5，我是不建议  
  - 因为就我了解的数据而言，稍微吃一点cpu的游戏表现都会相对弱  
  - 与此同时价格又是居高不下，我认为不如买xe3的10xe或者12xe
- 8040s、8050s、8060s  
  - 全能本u，优势在于四通道内存控制器  
    - 对于大显存、大内存带宽需求的东西合适，比如llm本地推理  
  - 如果你在考虑16cu的8040s，除非你明确自己想法或者它价钱确实低，不然不要考虑  
  - 32cu的8050s和40cu的8060s  
    - 前者大致4050，后者大致4060。我建议你在明确自己需求再考虑它们
